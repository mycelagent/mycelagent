**Mycel** is a local-first autonomous AI agent. It runs on your machine, persists memory across sessions, and uses tools (bash, files, web, memory) in a think → act → observe loop.

```
prompt → Think → Act → Observe → … → done

Tools: bash · read_file · write_file · list_dir · web_fetch · memory_save · memory_recall
```

```bash
git clone https://github.com/mycelagent/mycel.git
cd mycel && npm install
ollama pull llama3.2
npx tsx src/index.ts chat
```
